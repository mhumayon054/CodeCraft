CodeCraft Project – Complete Issues & Improvement Script
1. Backend Issues

a) Auth / JWT / Session Problems

Tumhara current backend passport-local aur express-session use kar raha hai, JWT nahi hai, aur session-based auth thik se kaam nahi kar rahi.

Errors console me ye aa rahe the:

GET http://127.0.0.1:5000/api/user 401 (Unauthorized)
POST http://127.0.0.1:5000/api/login 401 (Unauthorized)
POST http://127.0.0.1:5000/api/register 500 (Internal Server Error)


Breakdown:

401 → Frontend fetch me credentials: "include" missing hai → session cookie send nahi ho rahi

500 → Backend me storage.createUser ya database call error, ya invalid data (name/email/password missing)

JWT authentication not implemented → future authentication secure karne ke liye JWT zaroori

Fix / Improvement:

Replace session-based auth with JWT tokens

Frontend se har API call me token bhejna

Backend me /login, /register, /user endpoints properly JWT verify karen

3. CORS / Network Issues

Frontend (5173) → Backend (5000) cross-origin request properly credentials ke saath nahi bhej raha → 401 errors
Fix:

fetch("http://127.0.0.1:5000/api/user", {
  method: "GET",
  credentials: "include"
})


Backend me cors properly configure:

app.use(cors({
  origin: "http://localhost:5173",
  credentials: true
}));

4. Port / Running Issues

EADDRINUSE → port already use me tha

Correct fix: existing process kill karo ya backend port change karo
